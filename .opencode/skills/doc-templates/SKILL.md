# 公文文种模板库

## 一、请示

**适用场景**：向上级机关请求指示、批准

**模板框架**：
```
[发文机关]关于[事由]的请示

[主送机关]：

[背景依据段：3-5句，交代政策依据、实际情况和必要性]

现将有关事项请示如下：

一、[基本情况或需要说明的事项]

二、[具体请示事项，明确、单一]

妥否，请批示。

（联系人：[联系人]，联系电话：[电话]）

[发文机关]
[成文日期]
```

**写作要点**：
- 一文一事，不把多项不相关事项捆在一起请示
- 请示事项明确具体，不写"酌情处理""适当安排"等模糊表述
- 背景段要说明"为什么要做"，请示事项段说明"具体要批什么"
- 结尾用"妥否，请批示"或"当否，请批示"
- 原则上保留联系人附注

## 二、报告

**适用场景**：向上级汇报工作、反映情况、回复询问

**模板框架**：
```
[发文机关]关于[事由]的报告

[主送机关]：

[引言段：交代报告缘由或总体工作情况，3-5句]

现将有关情况报告如下：

一、基本情况（或：工作开展情况）
[交代起因、范围、现状和总体态势，数据支撑]

二、主要做法（或：取得成效）
（一）[做法/成效1]
（二）[做法/成效2]

三、存在问题（或：困难与不足）

四、下一步工作打算

特此报告。

（联系人：[联系人]，联系电话：[电话]）

[发文机关]
[成文日期]
```

**写作要点**：
- 报告中不夹带请示事项
- "基本情况"要交代清楚起因、范围、现状
- "主要做法"要落实到具体举措，有数据支撑
- "存在问题"要直陈短板、风险和制约因素
- "下一步打算"要落到后续安排和改进措施

## 三、通知

**适用场景**：发布、传达、转发事项或安排部署工作

**模板框架**：
```
[发文机关]关于[事由]的通知

[主送机关]：

[依据段：交代目的、依据，3-4句]

现将有关事项通知如下：

一、[总体要求或指导思想]

二、[重点任务或主要内容]
（一）[任务/内容1]
（二）[任务/内容2]

三、[工作要求或保障措施]

特此通知。

（联系人：[联系人]，联系电话：[电话]）

[发文机关]
[成文日期]
```

**写作要点**：
- 通知重在部署安排，要有可操作性
- "重点任务"要说清做什么、谁来做、何时完成
- "工作要求"要说清怎么保障、怎么督查
- 会议通知需写明时间、地点、参会人员、议程

## 四、函

**适用场景**：不相隶属机关之间商洽工作、询问和答复问题、请求批准

**模板框架**：
```
[发文机关]关于[事由]的函

[主送机关]：

[发函缘由：3-5句，交代背景和沟通目的]

现将有关事项函告如下（或：现就有关事项函请如下）：

一、[事项说明]

二、[具体要求或请求]

请予函复（或：特此函达、特此函复）。

（联系人：[联系人]，联系电话：[电话]）

[发文机关]
[成文日期]
```

**写作要点**：
- 函是平行文，语气要得体谦和，不用"要""必须"等命令性用语
- 去函（主动发函）结尾用"请予函复"
- 复函（回复来函）结尾用"特此函复"
- 函的附件要齐全

## 五、纪要

**适用场景**：记载会议主要情况和议定事项

**模板框架**：
```
[会议名称]纪要

时间：[年月日（星期X）上/下午X时]
地点：[会议地点]
主持人：[姓名职务]
出席：[姓名职务列表]
请假：[姓名职务列表]
列席：[姓名职务列表]

[正文]

[印发机关]
[成文日期]
```

**正文部分按以下提示语组织**：
- **会议认为**：总体判断和一致意见（写对形势的基本判断）
- **会议指出**：重要观点、基本情况、成绩、问题或意义（写对工作的定性）
- **会议强调**：需要重点重申的要求和定调（写必须守住的底线和方向）
- **会议要求**：直接可执行的任务、责任、时限和落实安排

**写作要点**：
- 客观记录，不添加会议之外的个人意见
- 议定事项要明确责任单位和完成时限
- 出席/请假/列席名单要准确
- 纪要一般不加盖印章

## 六、工作总结

**适用场景**：阶段性复盘、年度总结、专项工作总结

**模板框架**：
```
[单位名称][时间段]工作总结

[引言段：以"在XX指导下""在XX领导下"开头，概述总体成绩，5-7句]

一、主要工作及成效
（一）[方面1]：排比式小标题（如"聚焦XX，推动XX提质增效"）
一是……二是……三是……
（二）[方面2]
……

二、主要做法与经验
（一）[做法/经验1]
（二）[做法/经验2]

三、存在问题与不足
[客观陈述短板和不足，不用过度自贬，3-5个方面]

四、下一步工作打算
（一）[打算1]
（二）[打算2]

[发文机关]
[成文日期]
```

## 七、讲话稿

**适用场景**：领导讲话、会议发言、动员部署、总结点评

**写作要点**：
- 开篇敬语得体（根据场合用"同志们""各位代表""各位领导"）
- 正文按"为什么→是什么→怎么办"逻辑展开
- 多用排比句增强气势，段内用"一是二是三是"分点
- 结尾要有号召力和感染力（"让我们……"句式）
- 注意领导个人语言风格和习惯用语

## 八、简报/情况专报

**写作要点**：
- 简报遵循"先事实、后判断、再建议"的逻辑
- 标题直接点明核心信息
- 篇幅控制在 800-1500 字
- 语言高度凝练，用数据说话

## 九、工作方案

**模板框架**：
```
[单位名称][工作名称]实施方案

一、指导思想
二、工作目标
三、主要任务
四、实施步骤
五、责任分工
六、保障措施
```
