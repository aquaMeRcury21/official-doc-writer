# 公文写作技巧指南

## 一、公文语言四大原则

### 1. 准确
- 措辞恰当，概念准确
- 判断合乎逻辑，避免歧义
- 引用政策依据必须核对原文
- 数据必须真实可查，无来源的标注 [待核实]

**常见错误示例**：
| 错误写法 | 正确写法 | 原因 |
|---------|---------|------|
| 大约 200 人左右参加 | 约 200 人参加 | "大约"与"左右"重复 |
| 截止目前已完成 | 截至目前已完成 | "截至"才是规范表述 |
| 二〇〇六年出生 | 2006年出生 | 公文中用阿拉伯数字 |

### 2. 平实
- 语言朴实无华，不堆砌辞藻
- 避免溢美虚饰，用数据和事实说话
- 不写标语口号式的空话

**应当避免的虚饰表达**：
| 应删除/修改 | 改为 |
|-------------|------|
| 取得了举世瞩目的辉煌成就 | 取得了显著成绩 |
| 在英明正确领导下 | 在正确领导下 |
| 以排山倒海之势 | 删除 |
| 史无前例的伟大创举 | 重要创新举措 |

### 3. 简明
- 开门见山，直截了当
- 删繁就简，避免冗长
- 多用短句，少用长句
- 每段只说一层意思

**删繁就简技巧**：
| 冗余写法 | 简洁写法 |
|---------|---------|
| 对问题进行深入细致的研究分析 | 深入分析问题 |
| 进行了全面的检查工作 | 全面检查了 |
| 我们的同志要高度重视 | 各单位要高度重视 |
| 已经完成了任务目标 | 已完成任务 |

### 4. 庄重
- 使用规范书面语
- 不用口语、方言、网络流行语
- 语气得体，对上谦敬、对下严肃、对平级谦和
- 多用公文专用语

## 二、标题拟定六看

1. **是否准确恰当**——标题是否准确概括了全文主旨
2. **是否明确集中**——是否聚焦一个中心事项
3. **是否照应总题**——各级小标题是否围绕总标题展开
4. **是否合乎逻辑**——标题内部的逻辑关系是否成立
5. **是否具有艺术性**——排比是否匀称、对仗是否工整
6. **是否匀称和谐**——同一级标题的字数和结构是否基本一致

**标题拟定口诀**：事由要精准，文种要对路，字数要匀称，逻辑要通顺。

## 三、小标题拟定技巧（体制内实战）

### 排比式小标题（公文最常见写法）

**4+4 字对仗型**（适合工作总结、汇报材料）：
- 深化理论武装，筑牢思想根基
- 狠抓整改落实，严守政治规矩
- 创新宣传方式，提升传播效能
- 夯实基层基础，建强战斗堡垒

**5+5 字对仗型**（适合讲话稿、部署文件）：
- 聚焦主责主业，服务中心大局
- 坚持问题导向，补齐短板弱项
- 强化责任担当，狠抓工作落实

**动宾结构型**（适合通知、方案的任务标题）：
- 抓好理论学习
- 抓实问题整改
- 抓严作风建设
- 抓牢阵地管理

### 写作技巧

1. **先提炼关键词**：从内容中提炼 2-3 个核心动词（如：完善、健全、深化、强化、规范）
2. **再搭配宾语**：为每个动词搭配对应的宾语（如：完善机制、健全制度、深化改革）
3. **最后分组对仗**：把相近主题的标题放在一起，确保字数和结构一致

## 四、经验提炼四步法

### 第一步：罗列工作、整理分类
将一段时间内的所有工作罗列出来，按主题分类（如理论学习类、宣传引导类、意识形态类、队伍建设类等）

### 第二步：归纳亮点、提炼主题
从每一类工作中归纳出最突出的亮点做法，提炼为一句概括语

### 第三步：划分层次、打磨标题
将亮点概括语转化为排比式小标题，确保字数一致、结构对仗

### 第四步：严谨行文、反复修改
填充正文内容，用数据说话，反复打磨，直到每句话都见血见肉

## 五、常见错误对照表（80+ 项摘选）

### 易错用词

| 错误 | 正确 | 说明 |
|------|------|------|
| 截止 | 截至 | "截止"是不及物动词 |
| 必需 | 必须 | "必需"是形容词 |
| 制订 | 制定 | "制定"强调定下来 |
| 权利 | 权力 | 公文中多用"权力" |
| 启事 | 启示 | "启示"是启发指示 |
| 必须的 | 必要的 | 公文不写口语 |
| 其它 | 其他 | "其它"非规范用法 |
| 了望 | 瞭望 | 规范用字 |
| 帐目 | 账目 | 规范用字 |
| 参预 | 参与 | 规范用字 |
| 份内 | 分内 | 规范用字 |

### 易错格式

| 错误 | 正确 |
|------|------|
| 国办发[2026]5号 | 国办发〔2026〕5号 |
| 06月05日 | 6月5日 |
| 第05号 | 第5号 |

### 易错标点

| 错误 | 正确 |
|------|------|
| 一, 二, 三 | 一、二、三 |
| "通知" | 《通知》（引用文件时） |
| 一是, 二是 | 一是、二是 |
| 第1、2、3条 | 第一、二、三条（或第1、2、3条） |

## 六、公文专用语分类汇编

### 开头用语
- 表示依据：根据、按照、遵照、依照
- 表示目的：为、为了
- 表示缘由：鉴于、关于、由于
- 表示时间：兹、现、即将

### 承接/过渡用语
- 承上启下：现将……如下、特作如下通知、经研究决定
- 补充说明：同时、此外、另外

### 结尾用语
- 请示：妥否，请批示 / 当否，请批示 / 以上请示，请审批
- 报告：特此报告 / 以上报告，请审阅
- 通知：特此通知 / 请遵照执行
- 函（去）：请予函复 / 函请支持为盼 / 盼复
- 函（复）：特此函复 / 特此函达
- 批复：特此批复
- 通报：特此通报
- 通告：特此通告
- 决定：此令（命令类）/ 特此决定

### 期请用语
- 恳请、拟请、望予、请予、恭请

### 期复用语
- 妥否、当否、可否、是否妥当

### 经办用语
- 业经、业已、已将、已按、经……研究

### 征询用语
- 当否、妥否、是否可行、可否

### 谦敬用语
- 承蒙、不胜、为盼、为荷

## 七、政治话语体系密度构建技巧

体制内公文需保持适当的政治话语密度，以下是常用表达范式：

### 开篇常用范式
```
以习近平新时代中国特色社会主义思想为指导，
深入学习贯彻……（重要会议/讲话精神），
深刻领悟"两个确立"的决定性意义，
增强"四个意识"、坚定"四个自信"、做到"两个维护"。
```

### 工作成效常用表达
- 聚焦……，推动……走深走实
- 深化……，促进……提质增效
- 强化……，确保……落地见效
- 锚定……，实现……突破提升
- 紧盯……，狠抓……整改落实

### 存在问题常用表达
- 我们也清醒地认识到，……仍存在一些短板和不足
- ……仍需加强
- ……有待提升
- ……还需持续用力
- ……与上级要求相比还有差距

### 下一步打算常用表达
- 持续深化……，在……上实现新突破
- 着力强化……，推动……再上新台阶
- 进一步压实……责任，确保……落到实处
- 聚焦……精准发力，不断提升……水平

## 八、体制内写材料进阶经验

1. **先做减法再做加法**：初稿快速写出来，删掉废话，再补充实质性内容
2. **数据是骨头，语言是肉**：每一块成绩都要有数据支撑
3. **标题定生死**：领导看材料先看标题，标题写好了就成功一半
4. **排比是节奏**：公文的节奏感来自于排比句，但不要堆砌过度
5. **请示要说清"为什么要做、具体要做什么"**，报告要说清"做了什么、效果如何、下一步怎么办"
6. **"一是二是三是"先概述，后分点，同一自然段内写完**
7. **问题不回避、成绩不过头**：保持务实文风
8. **多模仿大领导的讲话稿**：学习其用词习惯和节奏感
