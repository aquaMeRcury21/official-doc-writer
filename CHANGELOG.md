# Changelog

## [0.1.0] - 2026-06-17

### Added

- Initial open-source release.
- AI-powered document drafting: notice, request, report, speech, briefing, work plan, meeting minutes, official letter, work summary.
- GB/T 9704-2012 compliant `.docx` generator (`utils/document_generator.py`).
- 9-dimension proofreading skill (`doc-proofread`).
- Three-layer TF-IDF RAG engine (`utils/rag_engine.py`).
- DeepSeek API client with retry, cost tracking, and budget enforcement (`utils/api_client.py`).
- Knowledge base auto-classification and archiving (`utils/knowledge_base_updater.py`).
- Multi-format document parser (`.txt`, `.docx`, `.pdf`, `.xlsx`).
- OpenCode integration: 8 skills, 14 slash commands, 3 agents.
